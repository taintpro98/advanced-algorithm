# BFS/DFS Algorithms

## What This Subtopic Is About

Breadth-First Search (BFS) and Depth-First Search (DFS) are fundamental graph traversal algorithms. This pattern includes standard traversals, state-space exploration, and variations with extended state tracking.

## Typical Patterns & Invariants

- **BFS**: Level-by-level exploration, shortest path in unweighted graphs
- **DFS**: Explore as deep as possible before backtracking
- **Multi-source BFS**: Starting from multiple nodes simultaneously
- **BFS on expanded state space**: Position + remaining resources
- **Bidirectional search**: Meet-in-the-middle approach

## How to Recognize This Type of Problem

- Graph or grid traversal
- Finding shortest path in unweighted scenarios
- Level-order processing
- State-space exploration with constraints
- Connected components, reachability problems

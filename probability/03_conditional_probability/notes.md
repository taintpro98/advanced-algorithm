# Conditional Probability & Bayes

## Core Concepts

- **Conditional probability**: P(A|B) = P(A ∩ B) / P(B)
- **Independence**: A and B are independent iff P(A ∩ B) = P(A) · P(B)
- **Law of Total Probability**: P(A) = Σ P(A|Bᵢ) · P(Bᵢ) over a partition {Bᵢ}
- **Bayes' Theorem**: P(B|A) = P(A|B) · P(B) / P(A)

## Key Insight

Bayes flips the conditioning: given evidence A, update belief about hypothesis B.
Prior P(B) → Likelihood P(A|B) → Posterior P(B|A)

## Classic Problems

- False positive paradox: rare disease, high-accuracy test → most positives are false
- Two-envelope problem
- Prosecutor's fallacy

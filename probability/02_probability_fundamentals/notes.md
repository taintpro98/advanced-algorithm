# Probability Fundamentals

## Core Concepts

- **Sample space** Ω: set of all possible outcomes
- **Event** A ⊆ Ω: a subset of outcomes
- **Probability axioms** (Kolmogorov):
  1. P(A) ≥ 0
  2. P(Ω) = 1
  3. P(A ∪ B) = P(A) + P(B) if A ∩ B = ∅

## Key Rules

- Complement: P(Aᶜ) = 1 - P(A)
- Inclusion-exclusion: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
- Union bound: P(A ∪ B) ≤ P(A) + P(B)

## Classic Problems

- Birthday problem: P(≥2 share birthday in n people) = 1 - 365!/((365-n)! · 365ⁿ)
- Monty Hall: switching wins with probability 2/3

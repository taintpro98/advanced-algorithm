# Probability & Mathematics

A structured study track for probability theory, statistics, and quantitative reasoning.
Separate from LeetCode — this section uses notes-only markdown format (no `.py` stubs).

## Structure

```
probability/
├── 01_counting_and_combinatorics/   ← permutations, combinations, pigeonhole
├── 02_probability_fundamentals/     ← sample spaces, axioms, classical problems
├── 03_conditional_probability/      ← Bayes' theorem, independence
├── 04_expectation_and_variance/     ← E[X], Var(X), linearity of expectation
├── 05_distributions/                ← Bernoulli, Binomial, Poisson, Normal, etc.
├── 06_random_walks/                 ← Markov chains, gambler's ruin, hitting times
└── 07_brainteasers/                 ← Interview-style puzzles and quant problems
```

## How to Use

Each subtopic has a `notes.md` with:
- Core theory and formulas
- Classic problems with full reasoning
- Interview-style questions to practice

This is a reading + reasoning track, not a coding track.
Reference `leetcode/practice/company_tags/worldquant/probability_and_brainteasers.md`
for additional quant-specific puzzles.

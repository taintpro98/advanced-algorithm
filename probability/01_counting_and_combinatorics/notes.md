# Counting & Combinatorics

## Core Concepts

- **Rule of Sum**: if A and B are disjoint, |A ∪ B| = |A| + |B|
- **Rule of Product**: if A and B are independent steps, |A × B| = |A| × |B|
- **Permutations**: ordered arrangements of k items from n → P(n,k) = n! / (n-k)!
- **Combinations**: unordered selections of k items from n → C(n,k) = n! / (k! × (n-k)!)
- **Pigeonhole Principle**: if n+1 items in n containers, at least one container has ≥ 2 items

## Key Identities

- Pascal's rule: C(n,k) = C(n-1,k-1) + C(n-1,k)
- Vandermonde: Σ C(m,k)·C(n,r-k) = C(m+n,r)
- Stars and bars: number of ways to put n identical items in k bins = C(n+k-1, k-1)
